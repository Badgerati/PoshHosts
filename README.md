# Hosts

Module that introduces a new `hosts` command on your terminal, that allows you to controls the hosts file from the command line - on Windows, Linux and MacOS.

## Install

Coming soon.

## Commands

Format: `hosts <command> [<value1>] [<value2>]`

```powershell
add
backup
clear
disable
enable
export
import
list
merge
path
remove
restore
set
test
```
